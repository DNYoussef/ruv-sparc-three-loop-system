# Skills Taxonomy Index

## Supporting Directories

- `skills/references/master-skills-index.md` – cross references, tags, trigger phrases, and status.  
- Category `_shared` folders – reusable examples or resources that apply to an entire category (for example `operations/observability/_shared`).  
- `_pipeline-automation/` – automation scripts, batch reports, and MECE validation logs.

---

## Working Notes

- When adding a new skill, place it inside the category that best matches its primary intent, update `skills/README.md`, and add the path to the relevant section above.  
- SOPs stay with their functional home (for example, QA-focused SOPs live under `quality/`).  
- If a skill has dependencies across categories, capture them in `skill.md` front matter and cross-link using relative paths.
