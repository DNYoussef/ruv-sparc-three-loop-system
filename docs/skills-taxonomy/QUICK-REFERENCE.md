## Glossary & Definitions

### Key Terms
- **MECE**: Mutually Exclusive, Collectively Exhaustive
- **Theater Code**: Code that looks good but doesn't work
- **SOP**: Standard Operating Procedure (defined process)
- **Swarm**: Multi-agent parallel execution
- **TDD**: Test-Driven Development

### Complexity Levels
- **Low**: Single function, <15 min, 1-3 agents
- **Medium**: Multi-function, 30-90 min, 3-5 agents
- **High**: Multi-domain, 2+ hours, 5+ agents

### Execution Models
- **Single-Agent**: One AI agent does the work
- **Dual-Agent**: Two agents coordinate
- **Swarm**: 5+ agents in parallel
- **Orchestrated**: Complex multi-skill pipeline
